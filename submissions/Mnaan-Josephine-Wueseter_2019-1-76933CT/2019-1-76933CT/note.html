Conceptual definition of web

  1. Hypertext information system: in hypertext definition system, instead of reading a text in rigid linear structure. One can skip or jump from one document to another(that is navigate, go back and front) simply by clicking on hot spots.

  2. Cross-platform: Ability to access information equally well from any computer hardware running from any operating system using any display.

  3. Interactive: It is the capability to talk back to the web server unlike traditional TV, that is ability to select a link, jump to another web page, and communicate with the publisher.

  4. Dynamic: This implies that the content can change any time even when a page is being viewed if not saved.

  5. Distributed: Distributed implies that the information provided is shared globally across millions of website. Each of which contributes the space for the information it publishes. These sites reside on one or more computers referred to as web servers.
