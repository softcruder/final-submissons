CONCEPTUAL DEFINITION OF A WEB

Web can be defined as:

  1. Hypertext information system.
  2. Cross platform.
  3. Interactive.
  4. Dynamic.
  5. Distributed.

  • Hypertext information system.: enables readers to use internet in a non-linear way.
    • Hypertext means instead of reading a text in a rigid linear way (Book) one can skip from one page to another.

  • Cross platform.: is the ability to access information equally well from any hardware running any operating system using any dispaly.

  • Interactive.: is the ability to talk back to the web browser unlike traditional television.
    • Ability to select a link, jump to another web page and communicate to the publisher of the web page

  • Dynamic.: Implies that the content can be changed anytime even if the page has being viewed is not saved.

  • Distributed.:Implies that the information provided is shared globally across million of website each of which contribute the space for the information it pubblishes.
