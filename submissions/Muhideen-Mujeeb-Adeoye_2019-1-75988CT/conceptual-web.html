Conceptual definition of web

Web can be defined into:

  • Hypertext Information System (Rigid Structure)
  • Cross Platform
  • Interactive
  • Dynamic
  • Distributed

Hypertext Information System (Rigid Structure)

Allow users to read in a rigid structure i.e jump from one page tp another

Cross Platform

Ability to access information equally from any computer running different operating system using any display

Interactive

Capability to talk back to the browsed web server unlike traditional TV

Dynamic

Implies that content can change anytime even when the pages is been view if not save

Distributed

Implies that information provided is share globally accross millions of website each of which contribute to space for the information published this site resides on one or more compute refers to as web server.
