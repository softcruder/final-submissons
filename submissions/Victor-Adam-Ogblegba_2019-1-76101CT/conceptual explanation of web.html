Conceptual explanation of a website

A website can be defined as a;

  1. Hypertext Information System
  2. Cross Platform
  3. Interactivity
  4. Dynamics
  5. Distributed

1. Hypertext Information System

Hypertext is a system for linking related text documents that allows the participation of multiple users. Hypertext doesn't allow you read a text in a rigid linear structure; one can skip from one point to another.

2. Cross Platform

Ability to access information equally well from any computer hardware running any operating system using any display unit. Such application are generally employ a cilent-server system architecture and vary widely in complexity and functionality.

3. Interactivity

Web interactivity refers to interactive features that are on websites that offer an exchange of information either between communication technology and users or between users using technology. Ability to select a link, jump to another webpage and communicate with publishers of the web pages.

4. Dynamics

A dynamic website contain information that changes depending on the viewer,the time of the day,the time zone,the viewer's native language and other factors.The content of dynamic website can change anytime even when the page is viewed if not saved.

5. Distributed

Implies that information provided is shared globally across millions of websites each of which contributes the space for the information it publishes. In distributed computing processing and data are spread out over multiple computers usually over a network.
