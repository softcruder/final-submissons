CONCEPTUAL DEFINITIONS OF A WEB PAGE

Conceptual definations of a good and effective web page are defined through these various qualities:

  • Hypertext Information System
  • Cross Platform
  • Interactive
  • Dynamic
  • Distributed

Hypertext Information System

Hypertext means instead of reading a text in a right linear structure e.g In a book,one can skip from point to another.Hypertext is a system for linking related documents that allows that allows the participation of multiple users.In a hypertext document,any word or phrase can be "hyperlinked" to information related to that word or phrase residing in the same document or in another document.A web page may contain texts,graphics,and hyperlinks to other web pages and files.

Cross Platform

Cross platform is the ability to access information equally well from any computer hardware running any operating system using any display. It can run on Microsoft, Windows, Linux and MacOS. This makes a webpage very flexibe and easy to access information with different Operating Systems and devices at thesame time.

Interactive

Interactivity is the capability for a web page to communicate and interact with users swiftly. This feature offer the user comprehensive, engaging, collaborative and amusing contents in the web page unlike the traditional television i.e the ability to select a link and jump to another web page and connect with the publisher.

Dynamic

Dynamic implies that the content can change anytime even when the page is being viewed if not saved. This means that the page displays different contents for different users while retaining thesame layout and design.

Distributed

Distributed implies that information provided is shared globally across millions of websites, each of which contributes the space for the information it publishes. Distributed web pages are transferrable, heterogenous, fault tolerant, scalable and flexibe. These sites reside on one or more computers referred to as web servers.
