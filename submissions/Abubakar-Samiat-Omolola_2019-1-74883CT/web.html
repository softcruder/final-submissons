Conceptual Definition Of Web

Conceptually,web is

  1. HyperText Information System : Means instead of reading a text in a rigid linear structure (i.e one can skip from one page to another).

  2. Cross-Platform: Ability to access information equally well from any computer hardware runninga any operating system using any display.

  3. Interactive: It is the capability to talk back to the web server unlike Traditional Television ability to select a link, jump to another web page and communicate with the publishers of the pages.

  4. Dynamic: Implies that the content can change anytime even when the page is being viewed if not saved.

  5. Distributed: Implies that information provided is shared globally across millions of website each of which contribute the space for the information it publishes. So,this site resides on one or more computer refers to as Web Servers.
