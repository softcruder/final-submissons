CONCEPTUAL DEFINTION OF WEB

Conceptually web is:

a. Hypertext Information System

b. Cross Platform

c. Interactive

d. Dynamic

e. Distributed

HYPERTEXT:

Instead of reading a text in a rigid linear structure (e.g a book) one can skip from point to another.

CROSS PLATFORM:

Ability to access information equally well from one any computer hardware running any operating system using a display.

INTERACTIVE:

Capacity to talk back to the web server unlike traditional TV i.e ability to select a link, jump to another web page and connect/communicate to the publisher of the pages.

DYNAMIC:

The content can change anytime even while the page is being viewed if not saved.

DISTRIBUTED:

Implies that information provided is shared globally accross millions of website, each of which contributes the space for the information it publishes. These sites resides on one or more computers refers to as web servers.
