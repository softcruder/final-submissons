THE CONCEPTUAL DEFINITION OF THE WEB

Conceptually, the Web is:

  • HYPERTEXT INFORMATION SYSTEM
  • CROSS-PLATFORM
  • INTERACTIVE
  • DYNAMIC
  • DISTRIBUTED

  1. HYPERTEXT INFORMATION SYSTEM

    2. Hypertext enables users read web pages in a nonlinear way (i.e. you can jump from page to page, or from one part of a page to another part of the same page). This means that instead of reading a text in a rigid/linear structure (e.g. a book), one can skip from one point to another.

  2. CROSS-PLATFORM

    3. This implies the ability to access information equally well from any computer hardware, running any Operating System, using any display, e.g. mobile devices, laptop, smart Tvs, Tablets, etc.

  3. INTERACTIVE

    4. Interactivity is the capability to talk back to the web server unlike with traditional TVs. It offers the ability to select a link, jump to another web page, and communicate with the publishers of the web page.

  4. DYNAMIC

    5. Dynamism implies that the content can change at anytime, even when the page is being viewed, if not saved.

  5. DISTRIBUTED

    6. This implies that information provided is shared globally across millions of websites, each of which contributes the space for the information it publishes. These sites reside on one or more computers referred to as web servers.
