Conceptual Definition of Web

Conceptually, web is:

  1. Hyper-text Information System
  2. Cross Platform
  3. Interactive
  4. Dynamic
  5. Distributed

1. Hyper-text Information System: Hypertext means instead of reading a text, in a rigid linear structure (e.g. in a book) one can skip from one point to another.

2. Cross platform: Ability to access information equally well from any computer hardware running any operating system using any display.

3. Interactive: The capability to talk back to web server unlike traditional TV i.e. ability to select a link, jump to another web page to connect and communicate with the publisher of the page.

4. Dynamic: Implies that the content can change any time even when the page is being viewed if not saved.

5. Distributed: Implies that information provided is shared globally across millions of websites each of which contributes the space for the information it publishes.
