conceptual definition of web

Conceptually : Web can be Define as;

  • Hypertext Information System
  • Cross platform
  • Interactive
  • Dynamic
  • Distributed

Hypertext Information System

Hypertext is a system for linking related text documents that allows the participation of multiple users. In a hypertexr document, any word or phrase can be "hyperlinked" to information related to that word or phrase residing in the same document or in another document.

Hypertext enables web user to use the web non-linearly. i.e, one can skip from one point to another.

Cross Platform

Ability to access information equally from any computer hardware from any running operating system using any display. e.g Using Mobile Phone, Laptop, desktop computer etc.

Interactiveness

Interactivity of a website is the capacity of a website to talk back to the web server unlike traditional TV. Ability to select a link, Jump to another web page and communicate to the publisher of the web page

Dynamic

Dynamic web pages implies that, the content can change anytime even when the page is being viewed if not saved

Distributed

Distributed web pages implies that, information provided is shared globally accross millions of Websites each of which contribute the space for the information it publishes.

These sites reside on one or more computers referred to as webservers
