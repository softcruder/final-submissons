CONCEPTUAL DEFINITION OF WEB




Web can be defined conceptually as a hypertext information system, it is crossplaform, interactive, dynamic and distributed.


  • Hypertext Information System: means instead of reading a text in a rigid linear structure(e.g in a book), one can skip from onepoint to another.

  • Cross platform: implies the ability to access information equally well from any computer hardware running from any hardware, running any os(Operating system) using any display.

  • Interactive: is the capability to talkback to the web server unlike traditional tv, ability to select the link, jumpout to another web-page and communicate to the publishers of the webpage.

  • Dynamic:implies that the content can change anytime even when the page is being filled, even if the page is being viewed if not saved.

  • Distributed: implies that the information provided is shared globally across million of websites each of whichcontributes its space for which information it publishes.
