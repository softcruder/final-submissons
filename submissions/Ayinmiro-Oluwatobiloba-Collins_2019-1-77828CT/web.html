Web can be defined conceptually as

  1. Hyper text information system
  2. Cross platform
  3. Interactive platform
  4. Dynamic
  5. Distributed

Hyper text allows one to jump from one chapter to another instead of reading sequentially chapter by chapter

Cross platform is the ability to access information equally well from any computer hardware running any operating system using any display

Interactvity is the capability to talk back to the web server unlike traditional TV that is jump to another web page and connect with the publisher of the page

Dynamic implies that the content can change anytime even when the page is being viewed if not saved

Distributed implies that information provided is shared globally across millions of websites each of which contributes the spce for the information it publishes, so these sites reside on one or more computers referred to as webservers

Technically,web is a collection of documents called web pages that are shared (for most part ) by computer users through out the world
