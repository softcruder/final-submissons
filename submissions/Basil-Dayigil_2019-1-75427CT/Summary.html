CONCEPTUAL DEFINITIONS OF A WEB PAGE

Conceptual definitions of a good and effective web page are defined through the following criterias:
    • Hypertext transformation system
      Cross Platform
      Interactive
      Dynamic
      Distributed

    • Hypertext Information System

    Hypertext means, Instead of reading a text in a right linear structure e.g in a book, one can skip from one point to another. Hypertext is a system for linking related documents that allows the participation of multiple users. In a hypertext document, any word or phrase can be "Hyperlinked" to information related to that word or phrase residing in the same document or in another document. A web page may contain texts, Graphics and hyperLinks to other web pages and files.

    Cross Platform

    This is the ability to access information equally well from any computer hardware running any operating system using any display

    Interactivity

    Interactivity is the ability to talk tp the web server unlike the traditional television i.e The ability to select a link and jump to another web page and connect with the publisher.

    Dynamic

    Dynamic implies that the content can change anytime even when the page is been viewed if not saved

    Distributed

    Distributed implies that the information provided is shared globally across millions of websites, each of which contributes the space for the information it publishes. These sites reside on one or more computers referred to as web servers
