Conceptual definition of a website

A website can be defined as a;

  1. Hypertext Information System
  2. Cross Platform
  3. Interactivity
  4. Dynamics
  5. Distributed

1. Hypertext Information System

Hypertext allows you to operate the internet in a non-linear structure. Hypertext doesn't allow you read a text in a rigid linear structure; one can skip from one point to another.

2. Cross Platform

Ability to access information equally well from any computer hardware running any operating system using any display unit

3. Interactivity

It is the capability to talk back to the users unlike traditional television. Ability to select a link, jump to another webpae and communicate with publishers of the web pages.

4. Dynamics

The content of can change anytime even when the page is viewed if not saved

5. Distributed

Implies that information provided is shared globally across millions of websites each of which contributes the space for the information it publishes. So these sites reside on one or more computers referred to as web servers.
