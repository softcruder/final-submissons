CONCEPT OF A WEB🌐


Conceptually, Web can be defined using the five parameters below:::
A. Hypertext Information System
B. Cross platform
C. Interactive
D. Dynamic
E. Distributed.

A. A Web can be defined as an Hpyertext Information System as it changes the crude method of accessing information from texts in a grid linear structure like in a book to hypertext (i.e you can skip from one point of the text to another with just a scroll) while accessing thesame information.

B. A Web due to its Cross platform nature has the ability to make access to information equally available to any user from any computer hardware running any operating system using any display.

C. A Web is Interactive, it has the capability to talk back to the web server, gives it user ability to select a link, jump to another webpage and even connect with the publishers if of the pages.

D. A Web is a Dynamic tool for Information sharing as the content of it can be changed and updated for credibility even when being viewed if not saved. This makes the Web an always current social tool.

E. A Web is Distributed in the sense that the Information provided is distributed globally across millions of website around the world each of which contributes the space for the information it publishes. This site resides on one or more computers refered to as Webserver.

Technically, Web is a collection of documents called webpages that are shared ( for most part) by computer users through out the world.

Internet is the collection of several billion computers connected through out the world.

Website is a collection of related webpages that are normally stored in a single webserver computer.

Webpage is a single html document that may contain text, graphics and icons.

Webserver is a computer system that enables users to access webpages stored in a webserver computer. Webserver is needed to publish pages on the web

Web Browser is the application used to view webpages such as this one and navigate the web. It is a piece of software that enables the user to retrieve and view a webpage.
