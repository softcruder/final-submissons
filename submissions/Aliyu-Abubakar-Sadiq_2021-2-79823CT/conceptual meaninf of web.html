CONCEPTUAL MEANING OF WEB

We have 5 conceptual meaning of Web

  1. Hypertext Information System:With web being hypertext makes it possible to read pages in non-linear and flexible form. You can read the first and choose to read the last one if you wish unlike a textbook which you can read linearly.
  2. Cross platform:For a web being cross platform means, in as much a gadget as an operating system and a display. Web can work on the gadget.
  3. Interactive:One of the difference between the web in the olden days and web nowadays web being interactive. In those days, you can only take information from the web, but nowadays you can get information from the web, you can talk back to the web server through the aspect of commenting.
  4. Dynamic:For a web or webpage to be dynamic means, the content can change anytime even when viewed(if not saved, bookmarked or downloaded).
  5. Distributed:The web is distributed because, the information provided is shared across millions of website with each of them contributing to that information.
