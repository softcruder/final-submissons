Conceptual definitions of WEB

  1. Hypertext Information System

  2. Cross Platform

  3. Interactive

  4. Dynamic

  5. Distributed

  1. Hypertext Information System

    2. Hypertext allows you to operate the internet in a non-linear structure. Hypertext doesn't allow you to read a text in a rigid linear structure.One can skip from one point to another

  2. Cross Platform

    3. This gives you the ability to access information equally well for any computer hardware running any operating system using any display.

  3. Interactive

    4. This is the capability to talk back to the web server unlike traditional tv. It's gives you the ability to select a link, jump to another web page and communicates with the publishers of the pages.

  4. Dynamic

    5. This implies that the contents can change anytime even when the page is been viewed if not saved.

  5. Distributed

    6. This implies that information provided is shared globally across millions of websites each of whcih contributes the space of the information it's publishes so this side resides on One or more computers referred to as Web servers.
