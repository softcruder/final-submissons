Web Development Explained

                            Web can be defined as;


  • Hypertext Information System
  • Cross Platform
  • Interactive
  • Dynamic
  • Distributed

    Hypertext:

    Instead of reading a text in a rigid unearly structure, Hypertext helps
    user read a website non-linearly

    Cross - platform:

    is ability to access information equally well from computer hardware
    running any operating system using any display

    Interactive:

    Interactivity is the capability to talk back to the server
    unlike traditional TV.  Ability to select a link, jump to another webpage and communicate with the publisters of the pages.

    Dynamic:

    Implies that the content can change anytime, even
    while the page is being viewed if not saved

    Distributed:

    Implies that the information provided is shared globally across millions of websites
    which contributes a space of which information is published.
    These sites resides on one or more computers reffered to as web servers.
