Conceptual Definition of The Web.

Conceptually, A Web is:

  1. HyperText Information System
  2. Cross-Platform
  3. Interactive
  4. Dynamic
  5. Distributed

HyperText Information System means instead of reading a text in a rigid linear structure, HyperText helps users navigate from one to another.

Cross-Platform is the ability to access information equally from any computer hardware running any operating system using any display.

Interactivity is the capability to talk back to the web server unlike traditional TV (i.e Ability to select a link, jump to another web page and communicate with the publisher of the pages.

Dynamic implies that the content can change anytime even when the page is being viewed if not saved.

Distributed implies that the information provided is shared globally across millions of websites, each of which contributes the space for which information it publishes. These sites resides on one or more computer referred to as web servers.
