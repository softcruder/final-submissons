Conceptually, Web is defined with 5 characteristics. They are;

  1. Hypertext Information System: With web being hypertext makes it possible to read pages in a non-linear and flexible form. You can read the first page and choose to read the last page if you wish. Unlike a textbook that you can read linearly.

  2. Cross-Platform: For web being a cross-platform means, in as much a gadget has an Operating system and a display. Web can work on such gadget.

  3. Interactive: One of the major difference between Web in the olden days and nowadays is web being interactive. In those days, you can only take information from the web, but nowadays as you get information from the web, you can talk back to the web server as well. through the aspect of commenting.

  4. Dynamic: For a web or webpage to be dynamic means, the contents of the page can change anytime even when the page is being viewed (if not saved, bookmarked or downloaded.)

  5. Distributed: The web is distributed because, the information provided is shared across millions of websites with each of them contributing to that information.
