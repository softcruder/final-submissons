CONCEPTUAL DEFINITION OF A WEBSITE

Conceptual definitions of a good and effective web page are defined through these various criteria:

  • Hypertext Information System
  • Cross Platform
  • Interactive
  • Dynamic
  • Distributed

Hypertext Information System

Hypertext means instead of reading a text in a right linear structure e.g a book,one can skip from point to the other. Hypertext is a system for linking related documents that allows that allows the use of multiple users. In a hypertext document, any word or phrase can be "hyperlinked" to information related to that word or phrase residing in the same document or in another document. A web page may contain texts,graphics and hyperlinks to other web pages and files.

Cross Platform

Cross platform is the ability to access information equally well from any computer hardware running any operating system using any display.

Interactive

Interactivity is the capability to communicate with the web server unlike the traditional television. It is the ability to select a link and jump to another web page and connect with the publisher.

Dynamic

Dynamic implies that the content can change anytime even when the page is being viewed if not saved.

Distributed

Distributed implies that information provided is shared globally across millions of websites,each of which contributes the space for the information it publishes.These sites reside on one or more computers referred to as web servers.
