CONCEPTUAL MEANING OF WEB

We have 5 conceptual meaning of Web

1. Hypertext Information System:   This enables users to use internet in a non-linearly means, instead of reading a text in a rigid linear structure(e.g Textbook) i.e One can skip from one point ot another.

2. Cross platform:   This means the ability to access information equally well form any computer hardware running operating system using any display.

3. Interactive:   This is the capability to talk back to the webserver unlike traditional TV i.e You have the ability to select a link, jump to another webpage, and communicate with the publishers of the pages.

4. Dynamic:   This implies that the content of the web can change anytime even when the page is being viewed if not saved.

5. Distributed:   This implies that information provided is shared globally across millions of website each of which contributes the space for the information it publishes. So these sites resides on one or more computer refered to as webserver.
