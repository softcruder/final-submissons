Conceptual Definition of the Web

Hypertext Information System: The web is based on the concept of hypertext, which is text that contains links to other documents or resources. These links allow users to easily navigate between different pages and access information from multiple sources.

Crossplatform: The wed has the ability to access information from any computer hardware running any operating system.

Interactive: The capability to talk back t the websever unlike traditiona TV, the user has the ability to select a link and jump from one webpage to another and communicate with the publishers of the webpages.

Dynamic: The contents of a webpage can be changed at any time even when the page is being viewed if not saved.

Disributive: This imlies that the information provided is shared globally across millions of websites.
